# 📱 Mobile Arena Web App

A Flask-based mobile shopping system with:
- Admin dashboard
- Customer filtering system
- Cart & order management
- Brand-based product management

## 🛠 Tech Stack
- Python (Flask)
- MySQL
- HTML, CSS

## 🚀 Features
- Filter phones by specs
- Add to cart
- Admin panel for managing products

```bash
python app.py
